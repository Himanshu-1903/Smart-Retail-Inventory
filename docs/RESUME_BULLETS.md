## For Software Engineering Roles

- **Built a full-stack Enterprise Inventory Management System** using Node.js, Express.js, MySQL 8, and vanilla JavaScript, managing 20 database tables with 60+ RESTful API endpoints
- **Designed and implemented a normalized 3NF relational database** with 20 tables, 6 stored procedures, 5 triggers, 5 analytical views, and a composite indexing strategy for optimal query performance
- **Implemented a layered backend architecture** (Route → Controller → Service → Repository) with dependency injection, transaction management, and custom error handling middleware
- **Developed role-based access control (RBAC)** with 3 roles, 25 granular permissions, JWT authentication with bcrypt hashing, and request-scoped permission caching for O(1) authorization checks
- **Built a real-time analytics dashboard** with canvas-based data visualization, KPI metrics, and business intelligence views including supplier performance scoring and product profitability analysis
- **Implemented transactional business workflows** including purchase order lifecycle management, sales order state machine with pessimistic locking (SELECT FOR UPDATE), and automated inventory tracking via database triggers

## For Database / DBMS Roles

- **Designed a production-grade MySQL 8 schema** in Third Normal Form with 20 tables, enforcing referential integrity through foreign key constraints and cascading rules
- **Created 6 stored procedures** for complex multi-table operations including PlacePurchaseOrder, ReceiveInventory, CreateSalesOrder, and MonthlySalesReport with cursor-based data processing
- **Implemented 5 database triggers** for automated business logic including real-time inventory updates on goods receipt, stock deduction on shipment, low-stock alert generation, and audit trail logging with JSON snapshots
- **Built 5 analytical SQL views** for business reporting: inventory valuation, supplier performance scoring, monthly revenue trends, product profitability, and customer purchase history aggregation
- **Designed a composite indexing strategy** covering 15+ indexes optimized for JOIN queries, range scans, and covering index patterns to minimize disk I/O

## For Full-Stack / Frontend Roles

- **Developed a single-page application dashboard** with dark glassmorphism UI design, responsive sidebar navigation, and 8 functional modules (Products, Inventory, Orders, Reports)
- **Built custom canvas-based data visualization** including gradient line charts for revenue trends and animated bar charts for product performance, with DPI-aware retina rendering
- **Implemented a centralized API client module** with JWT token management, automatic 401 redirect handling, and standardized request/response formatting
